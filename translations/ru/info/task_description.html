На этом острове Роботы нашли несколько загадочных коробок. После небольшого исследования они обнаружили интересные возможности этих коробок. Если поместить что-то в одну из них, то забрать это можно в любой из этих коробок. Стефан хочет использовать это для быстрого перемещения грузов, так как без груза он может перемещаться в два раза быстрей. То есть он может положить груз в ближайшую коробку, добежать до ближайшей к точке доставки и забрать там груз. Но на карте есть еще озера и реки, которые Стефан не может пересечь.

Карта местности представлена, как массив строк, где:

  • "W" - это вода (закрытая клетка)
  • "B" - коробка.
  • "E" - точка доставки.
  • "S" - стартовая точка.
  • "." - пустая клетка.

Стефан может перемещаться между соседними клетками за две минуты, если он несет груз. И без груза на это тратится только одна минута. Погрузить или выгрузить груз в коробку занимает одну минуту. Вам нужно найти быстрейший способ доставить груз.

Маршрут должен быть представлен строкой, где каждая буква обозначает действие.

  • "U" -- Верх (север)
  • "D" -- Вниз (юг)
  • "L" -- Лево (запад)
  • "R" -- Право (восток)
  • "B" -- загрузить или разгрузить груз в/из коробки.

Входные данные: Карта местности, как список (list) строк (str).

Выходные данные: Быстрейший маршрут, как строка (str).

Примеры: 

checkio(["S...","....","B.WB","..WE"]) #RRRDDD
checkio(["S...","....","B..B","..WE"]) #DDBRRRBD

Как это используется: Эта задача похожа на "Лабиринт", но с разной "стоимостью" движений. Это может быть полезно для расчета маршрута по местности с территориями имеющими различную проходимость.

Предусловия: 0 < rows < 10
0 < columns < 10
∀ x,y ∈ coordinates : 0 ≤ x,y ≤ 10
